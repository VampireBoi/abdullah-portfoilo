# Abdullah Homsi - Portfolio Website

A modern, dynamic portfolio website showcasing game development work and programming skills.

## Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool and dev server
- **TailwindCSS** - Styling
- **Framer Motion** - Animations
- **React Router** - Navigation

## Color Palette

- Dark Background: `#090a12`
- Secondary Dark: `#22203c`
- Muted Purple: `#87717d`
- Accent Pink: `#f0bbc2`
- Light/White: `#ffffff`

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Project Structure

```
src/
├── components/       # Reusable components
├── sections/        # Page sections (Hero, Work, Career, About)
├── pages/           # Route pages (Home, ProjectDetail)
├── data/            # Project data (JSON)
├── hooks/           # Custom React hooks
├── utils/           # Utility functions
└── styles/          # Global styles
```
